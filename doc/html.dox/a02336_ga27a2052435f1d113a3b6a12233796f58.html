subroutine mfisvr ( integer*8  fid,
character*(*)  version,
integer  cret 
)

Lecture du numéro de version de la bibliothèque MED utilisée pour créer le fichier (renvoyé sous la forme d'une chaîne de caractères).

Paramètres
fidIdentificateur du fichier.
versionNuméro de version défini dans une chaîne de caractères.
cretretour négatif en cas d'erreur (MED_ERR_HOWTO), Zéro sinon.
Voir également
MEDfileStrVersionRd

Cette routine lit le numéro de version de la bibliothèque MED qui a été utilisée pour créer le fichier auquel on accède via le paramètre fid. Le numéro de version est renvoyé sous la forme d'une chaîne de 9 caractères sous la forme MED-M.m.r. Exemple de chaîne renvoyé : 'MED-3.0.0'.
